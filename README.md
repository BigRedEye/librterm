=======


Minimalistic terminal emulator written on C++ and SDL2.

Building
--------
    cmake .
    make
    
Requirements
------------
+ [SDL2](https://www.libsdl.org/download-2.0.php)
+ [SDL2_ttf](https://www.libsdl.org/projects/SDL_ttf/)
+ [cmake](https://cmake.org/)
